실행 방법

0. FROG_WORLD.bak로 데이터베이스를 복원합니다. 
(odbc이름은 “frogworld_odbc” 입니다.)

1. 서버를 실행합니다. 실행에는 아래와 같은 파일들이 필요합니다.
- grassMap.txt 
- AI_SCRIPT\monster.lua

2. 클라이언트를 실행합니다. 실행에는 아래와 같은 파일들이 필요합니다.
- grassMap.txt
- res\... (여러 이미지 리소스 파일)

3. 클라이언트를 실행하면 아이피주소를 입력한 후 ID값을 입력합니다. 

4. 접속에 성공합니다. (입력한 ID값이 이미 접속 중이면 화면에 아무것도 나오지 않습니다.)

5. 더미클라이언트를 실행하면 id 0~499까지 접속합니다. 
(따라서, 만약 더미클라이언트를 먼저 실행하고 이후 클라이언트로 접속을 시도할 때는 id 500이상을 입력해야 중복되지 않습니다.)
